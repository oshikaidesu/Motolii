## B⑤ コード確認結果

`crates/motolii-doc/src/schema.rs` の `BlendMode` は通常の serde enum:

```rust
#[derive(Serialize, Deserialize)]
#[serde(rename_all = "snake_case")]
pub enum BlendMode {
    Normal,
    Add,
    Multiply,
}
```

- 未知値(例 `"blend": "screen"`)は **Deserializeエラー**になる(友人の「現serdeはエラー」は正しい)
- ただし **F-9違反とまでは断定できない**。F-9の未知フィールド保持は主にプラグイン拡張点の契約であり、コア閉集合enumの未知variantにもフォールバックを要求するかは設計判断
- B⑤で決めるべき選択肢:
  1. 未知BlendModeを**拒否**する。ただし「F-9の対象外」と明文化する
  2. `Unknown(String)`相当で保持し、描画時だけNormalへフォールバックする
- **避けるべき**: 単に serde default で Normal へ落とす(再保存で元の値を失う)
