# M2発注ゲート停止(2026-07-12)

ステータス: **採用**(プロセス訂正。実装PRではない)

## 判定

> M2は入場ゲートを丁寧に作ったが、既知領域であるジャーナル・Undo・音声を一行タスクのまま残し、スキーマ意味論が閉じる前に D1a〜c を完了扱いした。**D1d/D2/D3追補/D4/D5/M3 をこのまま発注するのは早い。**

「慎重に入場した」ことと「細目が閉じて発注できる」ことは**別種類のゲート**である。入場条件(M2E)全緑は Document に触る実装の入場であり、[M2仕様](../specs/M2-document-model.md)の確定ゲートを代替しない。仕様プロセス(ドラフト中・未決依存タスク禁止)に反して完了ラベルを付けた取り違えを、ここで撤回する。

正本のタスク表・未決ブロッカー: [M2-document-model.md](../specs/M2-document-model.md)。索引: [specs/README.md](../specs/README.md)。

## 批判12点の要約

1. **仕様プロセス破り**: M2を確定扱いし D1a〜c を完了・マージした一方、細目は未閉
2. **D1a/D1b完了は過大**: 重なり規則・グループ時間・拍0・Driver時刻・Asset指紋型・B群・複製ID再写像等が棚卸し未決のまま「スキーマ/validate完了」と表示([D1調査メモ](2026-07-12-d1-spec-holes-prior-art.md))
3. **D1d↔D2依存逆転**: ジャーナルが Command codec / マイグレーション接続なしで D1c だけに依存していた
4. **プロセス単一writer欠如**: 同一ファイル並行保存を last-write-wins で許す暫定が、journal 導入後は危険
5. **journal耐久契約欠如**: fsync/バッチ損失/ENOSPC/順序/checkpoint kill/破損種別が未定義。D1c完了条件の「ジャーナルから復元」は本体未実装と矛盾
6. **Undo完了条件が弱い**: `apply→revert`だけでは主要事故を捕まえない
7. **音声クロック内部矛盾**: デバイス主 × アンダーランでクロック停止+無音 / 供給済み≠聞こえている位置
8. **D5レンダ進捗が未定義**: フレーム並列で単調一値にならない。遷移パラメータ欠落
9. **D5完了条件が規約違反**: 「人間の耳」はレビュー項目であり自動判定完了条件にできない
10. **D6は gain と stream copy が両立しない**: 分岐と D1a/ExportJob 依存が必要
11. **D3依存グラフ未一本化**: 親循環だけ・LookAt/Followは存在確認のみ → 全体循環がすり抜ける
12. **Arcスナップショットが毎回 deep clone**: D8完了前に負荷試験が必要

## 即時停止対象

新規エージェント発注・「完了」ラベル付きマージを止める対象:

- D1d(ジャーナル)
- D2(Command / Undo)
- D3追補および **D3-dep**(統一依存グラフ)の新規発注
- D4 / D5 / D6
- M3(UI)の本発注

**完了表示の撤回**(再開扱い): D1a / D1b / D3(部分実装は残るが完了ではない)

**限定注釈**: D1c はスナップショット原子性に限り達成。ジャーナル復元・プロセスロックは含まない。

## 進行中 worktree

B⑤ BlendMode / Affine2 / stable ID / Composition.background 等の未マージ意味論作業は、**本ゲート再確認なしにマージしない**。仕様改訂(本メモ+ M2 ドラフト化)と実装コミットは分離する。

## 再開条件(優先順)

1. M2を正式にドラフトへ戻し、D1a/D1b(および過大なD3完了)を再開扱いにする → **本メモで実施**
2. Command 永続形式・transaction・stable ID・coalescing を仕様で確定
3. journal の durability / locking / checkpoint 状態機械を確定
4. audio clock / underrun / mode transition を状態機械として確定
5. D1d / D2 / D4 / D5 をそれぞれ複数PRへ再分割した発注書を書く
6. その後に D3-dep → D3追補、および M3 を解禁

再開時は M2 ステータスを再「確定」し、本メモを「解除」または「履歴」へ移す。
