## 取込規約

- ユーザー撮影画像はAMの版、OS、撮影日、操作直前の状態をmanifestへ記録する。
- アカウント名、通知、端末識別子等は取込前に除去する。
- 画像から観察できない内部実装、保存形式、全版での機能不在を推定しない。
- 公式ヘルプ画像は複製せず、URL、確認日、該当見出しを観察台帳へ記録する。
- 未取込の観察を実機確認済みとして完了扱いしない。
